# 04 — Components & Design System (Figma)

## Core components (with variants)
1) **Button**
   - Variants: Primary / Secondary / Tertiary
   - States: Default / Hover / Disabled / Loading
2) **Card**
   - Variants: Default / Selectable / Selected / Disabled
3) **Radio Group (1–5 scale)**
   - Horizontal scale with labels 1–5
   - Optional helper label row (Very Low → Excellent)
4) **Badge / Chip**
   - Variants: Status (Draft/Sent/Approved), Classification (Good/Needs Improvement/Clear Gap), Gap
5) **Progress Bar**
   - Used for element % and overall %
6) **Stepper / Breadcrumb**
   - Sector → Level → Assessment → Results → Recommendations → Approval → Report
7) **Table**
   - For results table in report: Element | Score % | Classification | Gap?
8) **Modal**
   - Confirm reset, or incomplete form warning
9) **Toast / Banner**
   - “Saved as Draft”, “Changes requested”, “Approved successfully”

## Typography (suggested)
- Title: 24–28
- Section header: 18–20
- Body: 14–16
- Caption: 12–13

## Spacing
- 8px base grid
- Card padding: 16–24
- Section spacing: 24–32

## Accessibility / UX
- Clear focus states
- Contrast-safe badges
- Always show scale meaning (1–5)
