# 01 — Problem & Scope

## Goal
Create a Figma prototype for a platform that:
1) Lets an evaluator select a **sector** and **job level**
2) Runs a **smart assessment form** (4 elements per level, 3 questions per element)
3) Calculates **element scores** + **overall score** and converts to **percentage**
4) Highlights **gaps**
5) Generates **recommendations** mapped to any gap
6) Provides **approval workflow** and **report issuance**
7) Stores report status: **Draft → Sent → Approved**

## In-scope sectors (cards)
- Education
- Healthcare
- Projects & Construction
- Administrative Services

## In-scope job levels and elements

### Executive
- Task completion
- Work quality
- Time commitment
- Procedure compliance

### Supervisory
- Task distribution
- Execution follow-up
- Solving daily problems
- Compliance with standards

### Managerial
- Work organization
- Report follow-up
- Coordination between departments
- Compliance with policies

### Leadership
- Setting objectives
- Team guidance
- Monitoring overall results
- Addressing challenges

## Rating scale (per question)
1 Very Low  
2 Low  
3 Medium  
4 Good  
5 Excellent

## Outputs
- Element result (percentage)
- Overall result (percentage)
- Classification:
  - Good
  - Needs Improvement
  - Clear Gap
- Recommendations if a gap exists
- Final report includes:
  - Sector
  - Job level
  - Evaluation results
  - Identified gaps
  - Suggested recommendations
  - Status: Draft / Sent / Approved
