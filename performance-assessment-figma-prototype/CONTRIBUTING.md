# Contributing

This repo is a **spec pack** for a Figma proof-of-concept prototype.

If you extend it:
- Keep frame names stable
- Document any new screens in `docs/03_screens.md`
- Add any new components to `docs/04_components_design_system.md`
- Update copy in `docs/06_copywriting_en.md` / `docs/07_copywriting_ar.md`
